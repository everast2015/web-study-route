# 2019年

## 2019年二月份

1. 读了《HTML5 与 CSS3 权威指南》这本书
2. 使用mpvue 将vue的项目转化为小程序的项目
3. 学习 SVG
4. 《图解CSS3》

> 关于《HTML5 与 CSS3 权威指南》这本书，确实给我了很大的启发还有就是感触，特别是标签的使用问题，之前的思维的方式就是一个标签只能使用一次，并且是在特定的场景中使用的，学习完之后，就打破之前的思维的方式，而已后续使用HTML5新增的语义化的标签的时候，将会更加的得心应手。
